### 💡 Sobre mí

Desarrollador **backend/full-stack** apasionado por crear soluciones escalables y eficientes. Me especializo en arquitectura backend robusta con **Java y Spring Boot**, combinado con experiencias frontend modernas en **React**. 

Mi filosofía: escribir código limpio, mantenible y bien documentado. Mi objetivo es transformar ideas en soluciones tecnológicas de alto impacto.

**🎯 Especialidades:**
- Arquitectura de microservicios y APIs RESTful robustas
- Desarrollo backend con Java & Spring Boot
- Interfaces modernas con React & TypeScript
- Bases de datos relacionales (PostgreSQL)
- Infraestructura con Docker & CI/CD

**🤝 Disponible para:** Colaboraciones, proyectos open-source y oportunidades de crecimiento
